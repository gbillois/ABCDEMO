Export

Choisissez d'abord un moteur PPTX éditable. Les anciens exports restent disponibles plus bas.

Nouveaux exports PPTX éditables

1. Export local JS — moteur local inspiré de l'ancien script Node, réécrit pour fonctionner directement dans cette page HTML. Aucun serveur, aucun Node à lancer.

2. Export DOM — utilise le bundle de la repo dom-to-pptx. Bon moteur de comparaison pour gradients, ombres, arrondis et SVG.

3. Export MagicSlider PPTX — pipeline intégré autour du moteur PPTXConverter, avec préparation HTML dédiée, styles PowerPoint-safe et validations de mise en page.
